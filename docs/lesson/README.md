# 课程目录

本目录是项目课程的主入口。每节课都是独立 Markdown 文件，包含学习目标、阅读文件、课堂内容、实践任务、检查问题和明确的课程完成条件。

## 学习顺序

建议按编号顺序学习：

1. [00-项目总览.md](00-项目总览.md)
2. [01-ROS2基础和项目运行方式.md](01-ROS2基础和项目运行方式.md)
3. [02-接口消息和数据语义.md](02-接口消息和数据语义.md)
4. [03-启动系统和参数系统.md](03-启动系统和参数系统.md)
5. [04-相机标定和TF.md](04-相机标定和TF.md)
6. [05-装甲板识别.md](05-装甲板识别.md)
7. [06-目标跟踪和EKF.md](06-目标跟踪和EKF.md)
8. [07-弹道解算和云台控制.md](07-弹道解算和云台控制.md)
9. [08-串口协议和下位机联调.md](08-串口协议和下位机联调.md)
10. [09-系统调试方法.md](09-系统调试方法.md)
11. [10-入门改造任务.md](10-入门改造任务.md)

## 使用方式

- 先完成 `00` 到 `03`，建立项目全局观。
- 再完成 `04` 到 `05`，跑通感知输入。
- 再完成 `06` 到 `07`，理解目标状态和控制输出。
- 最后完成 `08` 到 `10`，进入联调、排查和小范围改造。

每节课的“课程完成条件”是进入下一节课前的最低门槛。没有硬件时，可以用源码阅读、topic 设计、假消息或 rosbag 替代实机实验，但需要明确记录替代方式。
